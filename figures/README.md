# Appendix Figures

This folder contains the appendix figures referenced by the thesis document.

The images include:

- OpenVSP geometry setup captures for the fuselage, wings, canard, horizontal tail and vertical tail.
- FlightStream P3D to CCS conversion captures.
- Final FlightStream aircraft views.
- Complete exported aerodynamic result tables included as figure files.

These figures are provided as supplementary appendix material so that the thesis PDF can remain concise while preserving traceability of the modelling and simulation workflow.
