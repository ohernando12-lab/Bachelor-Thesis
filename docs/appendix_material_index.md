# Supplementary Appendix Material Index

This file lists the material currently prepared for the online supplementary appendix.

## Code

### OpenVSP

- `code/openvsp/openvsp_parametric_geometry_export_template.py`
  - Parametric geometry and Plot3D export template.

### FlightStream

- `code/flightstream/flightstream_scripts_template.txt`
  - Canard preprocessing template.
  - Canard solver setup and sweep template.
  - Conventional tail preprocessing template.
  - Conventional tail solver setup and sweep template.

### MATLAB

- `code/matlab/canard_geometry_trim_iteration_template.m`
  - Sensitivity-based geometry correction template.
- `code/matlab/flightstream_data_refinement_matlab_template.m`
  - FlightStream data cleaning and polar plotting template.
- `code/matlab/mesh_convergence_plot_generator_template.m`
  - Mesh-convergence figure and table generation template.

### VBA

- `code/vba/excel_data_refinement_vba_template.txt`
  - Excel data-refinement macro template.

## Figures

The `figures` folder is reserved for selected appendix figures that may be moved from the thesis PDF appendix to the online supplementary repository.

## Tables

The `tables` folder is reserved for complete processed tables or exported table images that are too large for the main thesis appendix.
